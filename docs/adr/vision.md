# Booking Platform

## Цель

Предоставить малому бизнесу простой сервис онлайн-записи.

## Клиенты

- Частные мастера
- Барбершопы
- Салоны красоты
- Массажные кабинеты

## Основные каналы

- Telegram
- Web
- Mobile (future)

## Принцип

Любой бизнес может зарегистрироваться,
создать компанию,
добавить сотрудников,
подключить Telegram
и начать принимать записи
без участия разработчика.
